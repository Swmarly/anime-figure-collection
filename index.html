    How to Customize

    1. Open figures.js and update each placeholder figure with your real data.
    2. Add new figures by copying an existing entry. The list renders itself automatically!
    3. Drop figure images in an assets/ folder (or link remote images) and update the image URL.
    4. Want more sparkle? Tweak colors or animations inside styles.css.
