      How to Customize

      1. Sign in to /admin to import figures from MyFigureCollection or add them manually. Changes are saved to Cloudflare KV and reflected on this page as soon as you reload.
      2. Want an offline backup? Use the download button in the admin panel to export the latest JSON snapshot directly from Cloudflare.
      3. Drop figure images in an assets/ folder (or link remote images) and update the image URL.
      4. Hosting the gallery separately from your Worker? Add data-api-base to the <html> tag (or set window.__FIGURE_COLLECTION_API_BASE__) so this page knows where to fetch the collection JSON.
      5. Want more sparkle? Tweak colors or animations inside styles.css.
